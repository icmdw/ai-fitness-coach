## 1. 设计风格

### 1.1 设计原则

- **简洁专业** — 清晰的视觉层次，减少不必要的装饰
- **数据驱动** — 突出数据展示，让用户快速了解训练进展
- **移动端优先** — 响应式设计，适配手机、平板、桌面
- **无障碍设计** — 符合 WCAG 2.1 AA 标准

### 1.2 配色方案

#### 主色调

| 颜色 | Hex | 用途 |
|------|-----|------|
| Primary Blue | `#2563EB` | 主按钮、链接、活跃状态 |
| Primary Dark | `#1E40AF` | 悬停状态、强调 |
| Primary Light | `#3B82F6` | 背景渐变、次要元素 |

#### 辅助色

| 颜色 | Hex | 用途 |
|------|-----|------|
| Success Green | `#10B981` | 完成状态、正面数据 |
| Warning Orange | `#F59E0B` | 警告、待完成 |
| Error Red | `#EF4444` | 错误、删除操作 |
| Info Purple | `#8B5CF6` | AI 功能、提示 |

#### 中性色

| 颜色 | Hex | 用途 |
|------|-----|------|
| Text Primary | `#1F2937` | 主要文字 |
| Text Secondary | `#6B7280` | 次要文字、标签 |
| Border | `#E5E7EB` | 边框、分割线 |
| Background | `#F9FAFB` | 页面背景 |
| Surface | `#FFFFFF` | 卡片、容器背景 |

### 1.3 字体

- **英文字体**: Inter, system-ui, -apple-system, sans-serif
- **中文字体**: 'PingFang SC', 'Microsoft YaHei', sans-serif
- **等宽字体**: 'Fira Code', 'Monaco', monospace（代码、数据）

#### 字体层级

| 层级 | 大小 | 字重 | 行高 | 用途 |
|------|------|------|------|------|
| H1 | 2rem (32px) | 700 | 1.2 | 页面标题 |
| H2 | 1.5rem (24px) | 600 | 1.3 | 模块标题 |
| H3 | 1.25rem (20px) | 600 | 1.4 | 子模块标题 |
| Body | 1rem (16px) | 400 | 1.5 | 正文 |
| Small | 0.875rem (14px) | 400 | 1.5 | 辅助文字 |
| Caption | 0.75rem (12px) | 400 | 1.4 | 图注、标签 |

## 3. 组件设计规范

### 3.1 按钮 (Button)

```tsx
// 变体
- primary: 主要操作（保存、提交）
- secondary: 次要操作（取消、返回）
- outline: 边框按钮（筛选、切换）
- ghost: 幽灵按钮（删除、更多操作）
- icon: 图标按钮（搜索、设置）

// 尺寸
- sm: 高度 32px, padding 8px 16px
- md: 高度 40px, padding 10px 20px (默认)
- lg: 高度 48px, padding 12px 24px

// 状态
- default
- hover
- active
- disabled
- loading
```

### 3.2 输入框 (Input)

```tsx
// 类型
- text: 文本输入
- number: 数字输入（重量、次数）
- date: 日期选择
- search: 搜索框（带搜索图标）

// 尺寸
- sm: 高度 32px
- md: 高度 40px (默认)
- lg: 高度 48px

// 状态
- default: 边框 #E5E7EB
- focus: 边框 #2563EB, 阴影 ring
- error: 边框 #EF4444, 错误提示
- disabled: 背景 #F3F4F6
```

### 3.3 卡片 (Card)

```tsx
// 结构
┌─────────────────────────┐
│ [Header] 可选            │
├─────────────────────────┤
│ [Content]               │
│                         │
└─────────────────────────┘

// 样式
- 背景：#FFFFFF
- 边框：1px solid #E5E7EB
- 圆角：8px
- 阴影：sm (hover: md)
- 内边距：16px 或 20px
```

### 3.4 表格 (Table)

```tsx
// 结构
┌─────────────────────────────────────────┐
│ [Header Row]                            │
├─────────────────────────────────────────┤
│ [Data Row 1]                            │
│ [Data Row 2]                            │
│ ...                                     │
└─────────────────────────────────────────┘

// 样式
- 表头背景：#F9FAFB
- 表头字重：600
- 行高：48px
- 边框：底部 1px solid #E5E7EB
- 悬停：背景 #F9FAFB
- 斑马纹：可选
```

### 3.5 标签 (Badge)

```tsx
// 变体
- success: 绿色背景，完成状态
- warning: 橙色背景，警告状态
- error: 红色背景，错误状态
- info: 蓝色背景，信息状态
- neutral: 灰色背景，默认状态

// 尺寸
- sm: 高度 20px, 字体 12px
- md: 高度 24px, 字体 14px
```

## 4. 响应式设计策略

### 4.1 断点定义

| 断点 | 最小宽度 | 设备类型 |
|------|----------|----------|
| xs | 0px | 小手机 |
| sm | 640px | 大手机 |
| md | 768px | 平板 |
| lg | 1024px | 小桌面 |
| xl | 1280px | 大桌面 |
| 2xl | 1536px | 超大桌面 |

### 4.2 布局适配

#### 移动端 (< 768px)
- 单列布局
- 底部导航栏
- 全屏表单
- 卡片堆叠
- 图表简化

#### 平板 (768px - 1024px)
- 双列布局（可选）
- 顶部或侧边导航
- 中等尺寸卡片
- 完整图表

#### 桌面 (> 1024px)
- 多列布局
- 侧边导航（可选）
- 大尺寸卡片
- 复杂图表与交互

### 4.3 组件响应式行为

| 组件 | 移动端 | 平板 | 桌面 |
|------|--------|------|------|
| 导航 | 底部栏 | 顶部栏 | 侧边栏 |
| 表格 | 卡片列表 | 滚动表格 | 完整表格 |
| 图表 | 单图表 | 双列 | 多列仪表板 |
| 表单 | 全宽 | 居中 | 居中/分栏 |

## 5. 交互设计规范

### 5.1 加载状态

- **骨架屏 (Skeleton)**: 用于内容加载
- **Spinner**: 用于按钮、小区域加载
- **进度条**: 用于长时间操作

### 5.2 空状态

```
┌─────────────────────────┐
│         📭              │
│     暂无数据            │
│  点击右下角 + 新增       │
│      [新增记录]         │
└─────────────────────────┘
```

### 5.3 错误状态

- 表单验证错误：即时显示，红色边框 + 错误文字
- 网络错误：Toast 提示 + 重试按钮
- 数据加载失败：空状态 + 重试按钮

### 5.4 成功反馈

- 表单提交：Toast 提示 + 自动跳转
- 数据保存：轻微动画 + 勾选图标
- 操作完成：Toast 或 Snackbar

## 6. 技术实现建议

### 6.1 UI 框架

- **React** + **TypeScript**
- **TailwindCSS** — 原子化 CSS，快速开发
- **Headless UI** 或 **Radix UI** — 无样式组件
- **自定义组件库** — 基于设计系统

### 6.2 图标库

- **Lucide React** 或 **Heroicons** — 一致风格
- 图标尺寸：16px, 20px, 24px

### 6.3 动画库

- **Framer Motion** — 页面过渡、微交互
- **TailwindCSS Animations** — 简单动画

### 6.4 表单处理

- **React Hook Form** — 高性能表单
- **Zod** — 表单验证
