## 1. 项目简介

### 1.1 功能概述

AI Fitness Coach 是一款个人健身力量训练和饮食管理的 Web 应用，帮助用户科学记录训练数据、追踪身体变化、获取 AI 辅助训练建议。

**四大核心功能**：

| 功能 | 说明 |
|------|------|
| 📝 训练记录 | 记录每次训练的动怍、组数、重量、次数 |
| ⚖️ 身体数据 | 追踪体重、体脂、围度等身体指标变化 |
| 📊 可视化统计 | 5 种图表展示训练进展和身体数据趋势 |
| 🤖 AI 助手 | 基于训练数据提供个性化训练建议 |

### 1.2 技术栈

| 层级 | 技术 |
|------|------|
| 前端 | React 18 + TypeScript + TailwindCSS + ECharts |
| 后端 | Node.js + Fastify + Prisma ORM |
| 数据库 | SQLite |
| 认证 | JWT Bearer Token |

## 2. 快速开始

### 2.1 用户注册

1. 访问应用首页，点击右上角 **注册** 按钮
2. 填写以下信息：
   - 姓名（用于个性化称呼）
   - 邮箱（用于登录）
   - 密码（至少 6 位）
3. 点击 **注册** 按钮
4. 注册成功后自动登录并跳转首页

### 2.2 用户登录

1. 访问应用首页，点击右上角 **登录** 按钮
2. 输入邮箱和密码
3. 点击 **登录** 按钮
4. 登录成功后 JWT Token 会自动存储，保持登录状态

### 2.3 首页导航

**桌面端**：顶部导航栏
- 首页 | 训练记录 | 身体数据 | 数据统计 | AI 助手 | 用户头像

**移动端**：底部导航栏
- 首页 | 训练 | 统计 | AI | 设置

## 3. 核心功能使用指南

### 3.1 训练记录管理

#### 创建训练记录

1. 点击 **训练记录** 页面
2. 点击右上角 **+ 新增记录** 按钮
3. 填写训练信息：
   - **训练日期**：选择训练日期
   - **训练类型**：胸部/背部/腿部/肩部/手臂/核心/全身/有氧
   - **添加动作**：搜索或输入动作名称（如：杠铃卧推）
   - **设置组数**：输入重量 (kg)、次数、组数
   - **备注**：记录训练感受或注意事项
4. 点击 **保存记录**

#### 查看训练历史

- 训练记录按日期倒序排列
- 每条记录显示：训练类型、日期、动作列表（重量×次数×组数）
- 点击记录可查看详情

#### 编辑训练记录

1. 找到要编辑的记录
2. 点击记录右上角 **编辑** 按钮
3. 修改训练信息
4. 点击 **保存**

#### 删除训练记录

1. 找到要删除的记录
2. 点击记录右上角 **删除** 按钮（红色垃圾桶图标）
3. 确认删除

### 3.2 身体数据记录

#### 记录身体数据

1. 点击 **身体数据** 页面
2. 点击右上角 **+ 新增记录** 按钮
3. 填写数据：
   - **测量日期**：选择日期
   - **核心数据**：体重 (kg)、体脂率 (%)、BMI、肌肉量 (kg)
   - **围度数据**（可选）：腰围、胸围、臂围、腿围 (cm)
   - **备注**：记录测量条件或感受
4. 点击 **保存记录**

#### 查看数据趋势

- 身体数据按日期倒序排列
- 每条记录显示与上次对比的 trend 图标（↑ 上升/↓ 下降）
- 最新数据在顶部概览卡片中展示

### 3.3 数据统计页面

统计页面提供 5 种可视化图表，帮助了解训练进展。

#### 力量进步曲线图

- **用途**：展示特定动作的最大重量随时间变化
- **操作**：点击动作标签切换（卧推/深蹲/硬拉等）
- **数据**：折线图显示重量趋势，标记最大/最小/平均值

#### 训练热力图

- **用途**：类似 GitHub 贡献图，展示训练频率和强度
- **颜色**：从浅到深表示训练强度（休息→低→中→高→极高）
- **交互**：鼠标悬停查看具体日期和训练量

#### 身体数据雷达图

- **用途**：多维度身体数据综合评估
- **维度**：体重、体脂率、肌肉量、BMI、腰围、胸围
- **对比**：当前值（蓝色）vs 目标值（绿色虚线）

#### 训练量柱状图

- **周训练量对比**：展示每周训练总量（kg）
- **肌肉群分布**：展示各肌肉群训练量分布

#### 目标完成度仪表盘

- **本周训练**：目标次数完成进度
- **本月训练量**：目标训练量完成进度
- **本月体重目标**：体重目标完成进度

### 3.4 AI 助手功能

#### 获取训练建议

1. 点击 **AI 助手** 页面
2. 在输入框输入问题（如："我最近力量停滞怎么办？"）
3. 点击 **发送** 或按回车
4. AI 会基于你的训练数据提供个性化建议

#### 查看建议历史

- 对话记录按时间顺序展示
- 可查看历史建议和分析报告

#### 常见问题示例

- "我应该如何安排每周训练计划？"
- "深蹲重量一直不增长怎么办？"
- "体脂率如何有效降低？"
- "训练后肌肉酸痛正常吗？"

## 4. 常见问题解答

### 4.1 账户相关问题

**Q: 忘记密码怎么办？**  
A: 目前版本需联系管理员重置密码。建议记录密码或使用密码管理器。

**Q: 可以修改用户名或邮箱吗？**  
A: 当前版本暂不支持修改，如有需要请联系管理员。

**Q: 登录状态能保持多久？**  
A: JWT Token 默认有效期 7 天，过期后需重新登录。

### 4.2 训练记录问题

**Q: 一次训练可以添加多少个动作？**  
A: 无限制，建议每次训练 3-6 个主要动作为宜。

**Q: 如何记录超级组或递减组？**  
A: 目前每个动作独立记录，可在备注中说明训练技巧。

**Q: 删除训练记录后能恢复吗？**  
A: 删除后无法恢复，请谨慎操作。

### 4.3 数据同步问题

**Q: 数据会自动保存吗？**  
A: 是的，每次操作都会实时同步到数据库。

**Q: 可以在多个设备上使用吗？**  
A: 可以，使用同一账号登录即可同步数据。

**Q: 数据会丢失吗？**  
A: 数据库定期备份，但建议定期导出重要数据。

### 4.4 技术支持

**Q: 发现 Bug 如何反馈？**  
A: 请记录以下步骤：
1. 问题描述
2. 操作步骤
3. 截图（如有）
4. 联系管理员反馈

**Q: 有新功能建议怎么办？**  
A: 欢迎提出！请说明功能场景和期望效果。
